The following code tranposes the element of an *M*Ã—*M* array, where *M* is a constant defined by `#define`:

```c
void transpose(long A[M][M]) {
    long i, j;
    for (i = 0; i < M; i++)
        for (j = 0; j < i; j++) {
            long t = A[i][j];
            A[i][j] = A[j][i];
            A[j][i] = t;
        }
}
```

When compiled with optimization level -O1, GCC generates the following code for the inner loop of the function:

```asm
.L6:
  movq    (%rdx), %rcx
  movq    (%rax), %rsi

  addq    $8, %rdx
  addq    $120, %rax
  cmpq    %rdi, %rax
  jne     .L6
```

We can see that GCC has converted the array indexing to pointer code.
